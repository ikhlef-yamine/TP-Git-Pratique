# Mini API Catalogue

Petite API REST pour gérer un catalogue de produits et de catégories.

## Installation

1. Cloner le dépôt GitHub.
2. Naviguer dans le répertoire : `cd mini-api-catalogue`
3. Installer les dépendances : `npm install`

## Lancement de l'API

Pour démarrer le serveur :

```bash
node server.js
# ou si vous utilisez nodemon : nodemon server.js
